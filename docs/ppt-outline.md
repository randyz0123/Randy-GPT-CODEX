# xiaofen.ai 对外 PPT 结构化大纲（11 页）

## 第 1 页｜封面
- xiaofen.ai 小粉 AI
- 私域销冠复制智能体
- 面向：私域销售团队管理者
- 目标：少漏客户、多成交、沉淀打法
- **视觉建议**：品牌主视觉 + 简洁标题
- **建议产品页**：`/`

## 第 2 页｜痛点页：为何需要现在行动
- 销冠依赖强，新人复制慢
- 跟进不一致，转化波动大
- 管理层看不到过程，难精细化管理
- 复盘耗时且难形成组织资产
- **视觉建议**：痛点矩阵（2x2）
- **建议产品页**：`/`（痛点区）

## 第 3 页｜价值页：我们提供什么
- 不替代销售，增强销售执行
- 下一步动作建议，降低漏单
- 销冠话术沉淀，提升团队可复制性
- 数据复盘驱动持续优化
- **视觉建议**：价值链路图（执行-复盘-优化）
- **建议产品页**：`/`（价值区）

## 第 4 页｜客户与行业适配页
- 行业：美妆快消、女装、大健康
- 组织：5~30 人私域团队优先
- 场景：高频咨询、高沟通密度、复购需求强
- 建议：先试点后扩面
- **视觉建议**：行业图标 + 角色画像
- **建议产品页**：`/`（行业区）

- **视觉建议**：左右分栏（输入/输出）
