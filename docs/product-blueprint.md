# xiaofen.ai 产品蓝图

## 产品愿景
打造“私域团队的 AI 销售操作系统”，让优秀销售能力从“个体天赋”变成“组织能力”。

我们追求的不是炫技，而是可量化的经营结果：
- 缩短新人上手周期
- 提升团队跟进一致性
- 稳定提高线索转化效率

## 用户画像

### 1) 决策者：创始人 / 业务负责人
关注点：投入产出比、团队可复制性、规模化增长。

### 2) 一线管理者：私域操盘手 / 销售主管
关注点：流程执行率、团队状态、转化漏斗诊断。

### 3) 执行者：销售顾问 / 私域运营
关注点：怎么说、何时跟、如何提高成交概率。

## 核心使用场景
1. **新人快速上岗**
   - 通过标准话术和场景建议，减少“不会聊、不敢推”的低效期。

2. **高意向客户临门转化**
   - 在异议处理与催单环节提供更稳妥的建议，提升关键节点转化。

3. **团队复盘与策略迭代**
   - 从沟通数据中识别共性问题，反向优化话术库和跟进 SOP。

4. **管理层过程管理**
   - 从“结果导向”扩展到“过程可视”，及时发现执行偏差。

## MVP 功能边界
MVP 明确“做什么 / 不做什么”：

### 做什么
- 话术知识沉淀与检索
- 场景化对话建议
- 基础跟进任务流
- 核心转化指标看板

### 不做什么
- 不直接接管所有客户沟通（保留人工决策）
- 不做重度 CRM 全量替代
- 不承诺“自动成交”或夸张增长

## 未来版本规划

### V1.1（数据增强）
- 更细粒度的客户标签体系
- 话术效果 A/B 对比
- 团队/个人能力画像

### V1.2（流程增强）
- 自动化触发器（如沉默客户提醒）
- 多渠道触点（企业微信、短信、邮件）
- 更完善的主管审批与质检机制

### V2.0（平台化）
- 行业模板市场（美妆、母婴、教育等）
- API 与第三方 CRM 深度集成
- 多组织、多品牌统一管理

## 商业化路径

### 收费模型
- 按席位订阅（基础版 / 专业版 / 企业版）
- 可叠加增值包（高级分析、行业模板、实施服务）

### 销售路径
1. 以“诊断 + Demo + 小范围试点”切入
2. 用 2~4 周试点验证关键指标（回复率、跟进及时率、成交率）
3. 试点达标后扩展到团队全量

### 交付策略
- 标准化产品 + 轻量顾问式落地
- 强调“上线速度快、学习成本低、可持续优化”
