# xiaofen.ai 商业化项目工作台

## 一句话定位
一个围绕「私域销冠复制智能体」的商业化推进工作台，用于把销冠能力沉淀为可复制、可扩张、可交付的成交系统。

## 目标客户
- 电商老板（尤其依赖私域成交与复购的团队）
- 私域操盘手与私域服务商
- 美妆快消品牌团队
- 女装品牌与直播电商团队
- 大健康行业的私域销售与转化团队

## 核心价值主张
- 不是多招一个销售，而是复制最强销冠的方法论与动作标准。
- 把个人经验变成团队可执行的成交流程，降低新人培养与管理成本。
- 用 AI 让话术、跟进、转化、复盘形成闭环，提升人效与成交稳定性。
- 先做可落地试点，再逐步扩展到多场景，确保投入产出清晰可衡量。

## 当前项目模块
- `docs/`：项目说明、产品与商业化方案、销售资料、客户案例沉淀
- `prompts/`：与项目相关的提示词资产（母提示词、销售提示词、内容生成提示词）
- `workflows/`：业务流程、AI 协作流程、客户交付 SOP、项目管理 SOP
- `assets/`：品牌素材、演示图片、截图与后续宣传物料

## 后续开发计划
1. **商业化资料标准化**：完善报价模板、试点方案模板、成交路径模板。
2. **产品能力清单化**：明确 MVP 能力边界，定义可交付与不可承诺范围。
3. **行业场景包建设**：围绕大健康、女装、美妆快消形成标准化场景方案。
4. **试点与复盘机制**：建立首批客户试点机制，输出可复制案例与复盘文档。
5. **协作流程产品化**：将关键流程沉淀为可执行 SOP，支持团队协同与规模交付。
